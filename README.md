# Housing Price Prediction — Linear & Logistic Regression From Scratch

## Overview
Implementation of Linear Regression and Logistic Regression from scratch using NumPy on the Housing dataset, without using sklearn model classes.

## Models Implemented
- **Linear Regression** — predicts house price (continuous)
- **Logistic Regression** — classifies price as high/low (binary)

## Techniques Used
- Z-score Standardization & Mean Normalization
- Gradient Descent (alpha = 0.01 and 0.1)
- MSE Cost Function & Cross-Entropy Cost Function
- Confusion Matrix Evaluation

## Dataset
Housing.csv contains features like area, bedrooms, bathrooms, furnishing status, etc.

## How to Run
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook
2. Upload `Housing.csv` when prompted
3. Run all cells in order

## Requirements
- Python 3
- NumPy, Pandas, Matplotlib, Seaborn
